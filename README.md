## 🛠️ Deploy Your Own Copy

### Prerequisites
- A GitHub account (free) — [github.com](https://github.com)
- Your phone or any computer with a browser

---

### Step 1 — Fork or Download

**Option A — Fork (easiest):**
1. Tap **Fork** at the top of this page
2. Tap **Create fork**
3. Skip to Step 3

**Option B — Fresh upload:**
1. Create a new repository named `deen-life` (Public)
2. Continue to Step 2

---

### Step 2 — Upload the File

1. In your repository, tap **Add file** → **Upload files**
2. Upload `index.html` (the app — single file, no dependencies)
3. Tap **Commit changes**

---

### Step 3 — Enable GitHub Pages

1. Go to your repository → **Settings**
2. Tap **Pages** in the left sidebar
3. Set **Source** → Branch: `main` → Folder: `/ (root)`
4. Tap **Save**

Your app goes live at:
```
https://YOUR-USERNAME.github.io/deen-life/
```
*(takes 2–3 minutes the first time)*

---

### Step 4 — Update in the Future

**From phone (GitHub website):**
1. Open `index.html` in your repo
2. Tap the ✏️ pencil icon → edit → **Commit changes**

**From computer (Git):**
```bash
git add .
git commit -m "Update description"
git push
```

## 🏗️ Tech Stack

- **Single HTML file** — zero dependencies, zero build tools
- **Vanilla JS** — no frameworks
- **IndexedDB** — offline Quran cache
- **CSS Variables** — obsidian / emerald / gold design system
- **PWA** — installable on Android & iOS

### APIs Used
| API | Purpose |
|-----|---------|
| [AlQuran Cloud](https://alquran.cloud/api) | Quran text (Arabic + translations) |
| [jsDelivr CDN](https://cdn.jsdelivr.net/gh/fawazahmed0/quran-api@1/) | Quran fallback CDN |
| [Quran.com API](https://api.quran.com) | Quran third fallback |
| [cdn.islamic.network](https://cdn.islamic.network) | Audio MP3 streaming |
| [Aladhan](https://aladhan.com/prayer-times-api) | Prayer times |
